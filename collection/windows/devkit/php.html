PHP

ZIP2026/5/21...小于 1 分钟

一、下载安装 PHP

  • 官网：https://www.php.net/downloads
  • 下载 Windows Thread Safe 版本的 ZIP（推荐 x64）
  • 解压到路径，例如：
C:\php

二、环境变量配置

  • 新建或编辑 系统 Path，添加：

    C:\php

  • 复制配置文件

    • 在 PHP 目录下，将：
    php.ini-development → php.ini

  • 关键配置（编辑 php.ini）

    • 设置扩展目录：
    extension_dir = "/path/to/ext"
    • 设置时区：
    date.timezone = "Asia/Shanghai"

三、验证 PHP

打开 CMD / PowerShell：

php -v

四、验证结果

正常输出示例：
