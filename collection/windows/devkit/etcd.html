etcd

ZIP2026/5/21...小于 1 分钟

一、下载安装 etcd

  • 官网：https://github.com/etcd-io/etcd/releases/
  • GitHub Release 下载 Windows 64-bit zip，例如：
etcd-v3.6.7-windows-amd64.zip
  • 解压到目录，例如：
C:\etcd

二、环境变量

  • Path 配置
    将解压目录加入系统 Path：
C:\etcd
  • 关键配置
    • 可创建 data 目录存储数据：
C:\etcd\data
  • 启动时可指定端口与数据目录：
etcd --data-dir C:\etcd\data --listen-client-urls http://127.0.0.1:2379 --advertise-client-urls http://127.0.0.1:2379

三、验证 etcd

打开 CMD / PowerShell，运行：

etcd --version
  • 启动 etcd：
etcd
  • 在另一个终端验证：
etcdctl endpoint status

四、验证结果

输出示例：

etcd Version: 3.6.7
  • etcdctl endpoint status 能显示节点状态，即表示运行成功
